What is ADMX Guide?

ADMX Guide is a free, independent reference platform for Windows Administrative Template (ADMX/ADML) policies. These policies are used by system administrators to configure security, behaviour, and features across Windows, Microsoft 365, browsers, and enterprise applications using Group Policy, Microsoft Intune, and other device management platforms.

The goal of this site is to make policy definitions easy to browse, understand, and implement — presenting them in a clean, searchable interface with direct visibility into their underlying registry settings.

Frequently Asked Questions

What are ADMX files?

ADMX files are XML-based Administrative Template files used by Windows Group Policy to define configurable settings for operating systems and applications. They replaced the older ADM format and are used with tools like the Group Policy Management Console (GPMC) and Microsoft Intune.

Who is ADMX Guide for?

ADMX Guide is designed for IT professionals, system administrators, security engineers, and anyone who manages Windows environments using Group Policy or Microsoft Intune. It is also useful for auditors who need to review policy configurations.

Is ADMX Guide affiliated with Microsoft?

No. ADMX Guide is an independent, community-built reference platform. It is not affiliated with, endorsed by, or sponsored by Microsoft or any other company whose products are referenced on this site.

How do I use these policies with Microsoft Intune?

Policies listed on ADMX Guide can be deployed via Intune using either the Settings Catalog or custom OMA-URI configurations. The registry path shown for each policy corresponds directly to the OMA-URI path used in Intune custom profiles.

How often is the content updated?

We aim to keep policy definitions up to date with the latest ADMX template releases from Microsoft and other vendors. If you spot an outdated or incorrect entry, please use our Contact page to let us know.
