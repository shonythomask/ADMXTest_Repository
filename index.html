What is ADMX Guide?

ADMX Guide is an independent reference platform for Windows Administrative Template (ADMX/ADML) policies. These policies are used by system administrators to configure security, behavior, and features across Windows, Microsoft 365, browsers, and enterprise applications using Group Policy, Microsoft Intune, and other device management platforms.

The goal of this site is to make policy definitions easy to browse, understand, and implement by presenting them in a clean, searchable interface with direct visibility into their underlying registry settings.
