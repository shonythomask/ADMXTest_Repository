About ADMX Guide

ADMX Guide is a free, independent reference platform built for IT professionals, system administrators, and anyone working with Windows Group Policy or Microsoft Intune.

The site provides a clean, searchable interface for browsing Administrative Template (ADMX/ADML) policy definitions across a wide range of enterprise applications — including Microsoft 365, Microsoft Edge, Google Chrome, Mozilla Firefox, and Citrix.

Why ADMX Guide Exists

Managing Group Policy in enterprise environments often means digging through raw ADMX files or scattered documentation to find the exact registry path or setting you need. ADMX Guide was created to solve that — putting all of that information in one place, presented clearly and consistently.

Whether you're configuring security baselines, locking down browser settings, or auditing existing policies, ADMX Guide helps you find what you need quickly.

What You Can Do Here

🔍 Browse Policies

Explore all available ADMX policies for each supported application, organised by category.

⚙️ View Registry Paths

See the underlying registry keys and values for each policy — essential for Intune and manual deployments.

🔎 Search Settings

Quickly find a specific policy by name or keyword without scrolling through long lists.

📋 Understand Options

View supported values, data types, and descriptions to understand exactly what each setting does.
